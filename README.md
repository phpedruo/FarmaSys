## 📝 Sobre o Projeto

O **FarmaTec** é um Sistema web para gerenciamento de medicamentos, controle de estoque e registro de vendas, permitindo acompanhar validade dos produtos e histórico de compras de clientes. O sistema auxilia no controle interno da farmácia, garantindo organização, rastreabilidade e segurança nas vendas.
